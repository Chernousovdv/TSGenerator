Модуль генерации синтетических многомерных временных рядов для предобучения модели TokDiT.

## Быстрый старт

### 1. Профилирование генератора

```bash
cd ts_generator
python config_calibration.py
```

**Что делает:**
- Генерирует 2000 синтетических рядов
- Вычисляет одномерные статистики (12 метрик)
- Вычисляет многомерные метрики (сигнатуры, кросс-корреляции, площади Леви)

**Выходные файлы:**
- `results/generator/run_X/profile_data.csv` — одномерные статистики
- `results/generator/run_X/` — графики распределений по уровням (latent/observed)
- `results/generator/multivariate_metrics.csv` — агрегированные многомерные метрики
- `results/generator/multivariate_metrics_raw.csv` — сырые многомерные данные

---

### 2. Анализ реальных данных (Monash)

```bash
# Режим 1: Сбалансированная выборка (рекомендуется для калибровки)
python analyze_monash.py diverse

# Режим 2: Финансовые и бизнес-датасеты
python analyze_monash.py financial

# Режим 3: Все доступные датасеты
python analyze_monash.py all
```

**Режимы работы:**

| Режим | Команда | Датасеты | Время | 1D сэмплов | MV сэмплов (B×D) |
|-------|---------|----------|-------|------------|------------------|
| `diverse` | `python analyze_monash.py diverse` | 13 датасетов (6 категорий) | ~3-5 мин | 1200 рядов | 100 × 4 = 400 |
| `financial` | `python analyze_monash.py financial` | 13 датасетов (финансы/бизнес) | ~5-7 мин | 1161 рядов | 200 × 4 = 800 |
| `all` | `python analyze_monash.py all` | 17 датасетов (все типы) | ~10-15 мин | ~1700 рядов | 100 × 4 = 400 |

**Примечание:** 
- **1D сэмплов** — число отдельных временных рядов для вычисления одномерных статистик (ADF, энтропия, сезонность и др.)
- **MV сэмплов** — число батчей (B) × размерность (D=4) для многомерных метрик (сигнатуры, кросс-корреляции, площади Леви)

**Состав режимов:**

**diverse** (сбалансированная выборка):
- Финансовые: bitcoin, fred_md
- Бизнес/Спрос: car_parts, pedestrian_counts
- Туризм: tourism_yearly, tourism_monthly
- Энергетика: australian_electricity_demand, london_smart_meters
- Погода: weather, saugeenday
- Трафик: traffic_weekly
- Прочее: hospital, sunspot

**financial** (финансовые и бизнес-датасеты):
- Криптовалюты: bitcoin
- Макроэкономика: fred_md
- Бизнес (M4): m4_yearly, m4_quarterly, m4_monthly, m4_weekly, m4_daily, m4_hourly
- Банковские (NN5): nn5_daily, nn5_weekly
- Туризм: tourism_yearly, tourism_quarterly, tourism_monthly
- Спрос: car_parts, pedestrian_counts

**all** (все доступные датасеты):
- Включает все вышеперечисленные + высокочастотные датасеты (traffic_hourly, solar_10_minutes, wind_farms, и др.)

**Что делает:**
- Загружает выбранные датасеты Monash через HuggingFace
- Вычисляет те же одномерные статистики (12 метрик)
- Вычисляет многомерные метрики на тех же данных

**Выходные файлы (для каждого режима):**
- `results/monash_profile/<mode>/monash_all.csv` — одномерные статистики
- `results/monash_profile/<mode>/monash_all.png` — графики распределений
- `results/monash_profile/<mode>/multivariate_metrics.csv` — многомерные метрики
- `results/monash_profile/<mode>/multivariate_metrics_raw.csv` — сырые многомерные данные

**Важно:** ОДИН и тот же набор датасетов используется для одномерных и многомерных статистик — это обеспечивает согласованность сравнения.

## Одномерные статистики

Вычисляются для каждого ряда отдельно. Всего 12 метрик:

| Статистика | Описание | Интерпретация |
|------------|----------|---------------|
| `mean` | Среднее значение | Для нормализованных данных $\approx 0$ |
| `std` | Стандартное отклонение | Для нормализованных данных $\approx 1$ |
| `acf_lag1` | Автокорреляция (лаг 1) | Близко к 1: сильный тренд; близко к 0: белый шум |
| `acf_lag2` | Автокорреляция (лаг 2) | Дополнительная мера инерции процесса |
| `adf_ct` | ADF тест (c trend) | Отрицательное: стационарность; положительное: нестационарность |
| `perm_ent_m3` | Пермутационная энтропия ($m=3$) | Близко к 1: хаотичность; близко к 0: регулярность |
| `mk_z` | Mann-Kendall $Z$ | Положительное: восходящий тренд; отрицательное: нисходящий |
| `seasonality` | Сила сезонности | $[0,\,1]$, где $1$ = сильная сезонность |
| `forecastability` | Предсказуемость | $[0,\,1]$, где $1$ = легко предсказывать |
| `fft_mean` | Средняя частота (FFT) | Близко к 0: низкочастотный процесс |
| `roughness` | Шероховатость | Мера изменчивости соседних точек |
| `trend_str` | Сила тренда | $[0,\,1]$, где $1$ = сильный тренд |

## Многомерные статистики

Вычисляются для батчей вида $(B,\,T,\,D)$ — $B$ рядов размерности $D$.

Многомерные метрики улавливают межканальные зависимости и сложные паттерны, которые невозможно обнаружить в одномерном анализе.

---

| Категория | Метрика | Размерность | Интерпретация |
|-----------|---------|-------------|---------------|
| **Сигнатуры** | `signature_wasserstein` | $K$ | Расстояние Вассерштейна между распределениями сигнатур |
| | `signature_mmd_simple` | скаляр | $\mathrm{MMD}^2$: квадрат разности средних сигнатур |
| | `signature_mmd_kernel` | скаляр | Ядерный MMD с линейным ядром |
| | `signature_energy_k` | `depth` | Энергия $k$-го уровня: $E_k = \lVert S_k \rVert^2$ |
| | `signature_entropy` | скаляр | Энтропия: $H = -\sum p_i \log p_i$, мера сложности |
| | `brownian_distance` | скаляр | Отклонение от броуновского движения |

| **Площади Леви** | `levy_area_k` | $D(D{-}1)/2$ | Антисимметричная часть уровня 2 |

| | `rotation_number` | скаляр | Число вращения: $\mathrm{RN} = \frac{1}{\pi}\sqrt{\sum (A^{i,j})^2}$ |

### 1. Сигнатуры (Signature)

**Математическое определение:**

Сигнатура — это последовательность итерированных интегралов вдоль траектории $X\colon [0,T] \to \mathbb{R}^D$:

$$S^{i_1,\ldots,i_k}(X) = \int_{0 < t_1 < t_2 < \cdots < t_k < T} dX^{i_1}_{t_1}\, dX^{i_2}_{t_2}\, \cdots\, dX^{i_k}_{t_k}$$

**Структура по уровням:**

| Уровень | Компоненты | Что кодирует | Пример для $D=3$ |
|---------|------------|--------------|------------------|
| 1 | $S^1,\, S^2,\, \ldots,\, S^D$ | Средние приращения (тренды) | 3 компоненты |
| 2 | $S^{11},\, S^{12},\, \ldots,\, S^{DD}$ | Ковариации, взаимодействия | 9 компонент |
| 3 | $S^{ijk}$ | Высшие порядки (нелинейности) | 27 компонент |

**Размерность:**

- **Полная сигнатура:**

$$K = \sum_{k=1}^{\text{depth}} D^k = D + D^2 + D^3 + \cdots + D^{\text{depth}}$$

- **Лог-сигнатура** использует формулу Витта для размерности свободной алгебры Ли:

$$\dim(L_k) = \frac{1}{k}\sum_{d \mid k} \mu\!\left(\frac{k}{d}\right) D^d$$

**Пример размерностей:**

| $D$ | depth | Полная | Лог-сигнатура |
|-----|-------|--------|---------------|
| 2 | 2 | $2 + 4 = 6$ | $2 + 1 = 3$ |
| 3 | 2 | $3 + 9 = 12$ | $3 + 3 = 6$ |
| 4 | 2 | $4 + 16 = 20$ | $4 + 6 = 10$ |
| 5 | 3 | $5 + 25 + 125 = 155$ | $5 + 10 + 35 = 50$ |


**Метрики на основе сигнатур:**

| Метрика | Формула | Диапазон | Интерпретация |
|---------|---------|----------|---------------|
| `signature_wasserstein` | $W\bigl(S_{\text{real}},\, S_{\text{synth}}\bigr)$ | $[0,\,+\infty)$ | Расстояние между распределениями |
| `signature_mmd_simple` | $\bigl\lVert \mathbb{E}[S_{\text{real}}] - \mathbb{E}[S_{\text{synth}}] \bigr\rVert^2$ | $[0,\,+\infty)$ | Квадрат разности средних |
| `signature_mmd_kernel` | $\mathbb{E}[k(X,X')] - 2\,\mathbb{E}[k(X,Y)] + \mathbb{E}[k(Y,Y')]$ | $[0,\,+\infty)$ | Ядерный MMD (линейное ядро) |
| `signature_energy_k` | $E_k = \lVert S_{\text{level}\,k} \rVert^2$ | $[0,\,+\infty)$ | Энергия $k$-го уровня |
| `signature_entropy` | $H = -\sum p_i \log p_i$, где $p_i = \lvert S_i\rvert / \sum \lvert S_j\rvert$ | $[0,\,\log K]$ | Мера сложности/хаотичности |
| `brownian_distance` | $\lVert S(X) - S(\mathrm{BM}) \rVert$ | $[0,\,+\infty)$ | Близость к броуновскому движению |
| `gram_eigenvalues_k` | $\lambda_k\bigl(S \cdot S^\top\bigr)$ | $[0,\,+\infty)$ | Эффективная размерность паттернов |

### 2. Кросс-корреляции (Cross-Correlation)

**Математическое определение:**

Для многомерного ряда $X \in \mathbb{R}^{T \times D}$ кросс-корреляция между каналами $i$ и $j$ при лаге $\tau$:

$$\rho_{i,j}(\tau) = \mathrm{Corr}\bigl(X_i(t),\; X_j(t+\tau)\bigr) = \frac{\mathrm{Cov}\bigl(X_i(t),\; X_j(t+\tau)\bigr)}{\sigma_i \,\cdot\, \sigma_j}$$

где $\tau \in [-\tau_{\max},\; +\tau_{\max}]$.

**Выходные данные:**

| Метрика | Размерность | Описание |
|---------|-------------|----------|
| `cross_corr_mean` | $(B,\,1)$ | Средняя абсолютная кросс-корреляция по всем парам и лагам |
| `cross_corr_by_lag` | $(B,\,D,\,D,\,2\tau{+}1)$ | Полная матрица корреляций для каждого лага |
| `cross_corr_wasserstein` | скаляр | Расстояние Вассерштейна между распределениями |

**Интерпретация по лагам:**

| Лаг $\tau$ | Интерпретация |
|------------|---------------|
| $\tau = 0$ | Синхронная связь между каналами $i$ и $j$ |
| $\tau > 0$ | Канал $i$ ведёт, канал $j$ запаздывает на $\tau$ |
| $\tau < 0$ | Канал $j$ ведёт, канал $i$ запаздывает на $\lvert\tau\rvert$ |

**Пример:** Для $D=4$, $\tau_{\max}=5$:
- Полная матрица: $(4,\,4,\,11) = 176$ значений
- Усреднённая: 1 значение на батч

### 3. Ранг корреляционной матрицы (Correlation Rank)

**Математическое определение:**

Для ряда $X \in \mathbb{R}^{T \times D}$ вычисляется корреляционная матрица $\mathbf{R} \in \mathbb{R}^{D \times D}$:

$$\mathbf{R} = \mathbf{D}^{-1/2}\,\mathrm{Cov}(X)\,\mathbf{D}^{-1/2}$$

где $\mathbf{D} = \mathrm{diag}\bigl(\mathrm{Cov}(X)\bigr)$ — диагональная матрица дисперсий.

**Метрики:**

| Метрика | Формула | Диапазон | Интерпретация |
|---------|---------|----------|---------------|
| `rank_ratio` | $\mathrm{rank}(\mathbf{R})\,/\,D$ | $[0,\,1]$ | Доля линейно независимых каналов |
| `condition_number` | $\lambda_{\max}(\mathbf{R})\,/\,\lambda_{\min}(\mathbf{R})$ | $[1,\,+\infty)$ | Чувствительность к шуму |
| `effective_rank` | $\exp\!\bigl(-\sum p_i \log p_i\bigr)$, где $p_i = \lambda_i / \sum \lambda_j$ | $[1,\,D]$ | Энтропийный ранг |

**Интерпретация:**

| Значение | `rank_ratio` | `condition_number` | `effective_rank` |
|----------|--------------|--------------------|------------------|
| **Идеально** | $\approx 1$ | $1$–$10$ | $\approx D$ |
| **Хорошо** | $> 0.8$ | $10$–$100$ | $> 0.7 D$ |
| **Плохо** | $< 0.5$ | $> 1000$ | $< 0.3 D$ |

**Когда `rank_ratio` $\approx 1$:**
- Все каналы линейно независимы
- $T \gg D$ (достаточно наблюдений)
- Нет мультиколлинеарности

**Когда `rank_ratio` $< 1$:**
- Есть линейные зависимости между каналами
- $D > T$ (ультрамногомерный режим)
- Почти вырожденная корреляционная матрица

### 4. Площади Леви (Lévy Area)

**Математическое определение:**

Площадь Леви для пары каналов $(i,\,j)$ — это антисимметричная часть сигнатуры уровня 2:

$$A^{i,j} = \frac{1}{2}\bigl(S^{i,j} - S^{j,i}\bigr) = \frac{1}{2}\sum_{r < s}\bigl(dX^i_r \cdot dX^j_s - dX^j_r \cdot dX^i_s\bigr)$$

**Геометрическая интерпретация:**

Площадь Леви измеряет площадь, «заметаемую» траекторией в плоскости $(i,\,j)$.

**Метрики:**

| Метрика | Формула | Размерность | Интерпретация |
|---------|---------|-------------|---------------|
| `levy_area_k` | $A^{i,j}$ для пары $k$ | $D(D{-}1)/2$ | Площадь для конкретной пары |
| `total_levy_area` | $\sum_{i<j} \lvert A^{i,j}\rvert$ | скаляр | Суммарная площадь |
| `levy_area_wasserstein` | $W(A_{\text{real}},\, A_{\text{synth}})$ | скаляр | Расстояние между распределениями |
| `rotation_number` | $\mathrm{RN} = \frac{1}{\pi}\sqrt{\sum_{i<j}(A^{i,j})^2}$ | скаляр | Среднее число «оборотов» |

**Интерпретация:**

| Значение | Интерпретация |
|----------|---------------|
| $A^{i,j} \approx 0$ | Траектория не имеет вращения в плоскости $(i,\,j)$ |
| $A^{i,j} > 0$ | Вращение против часовой стрелки |
| $A^{i,j} < 0$ | Вращение по часовой стрелке |
| $\lvert A^{i,j}\rvert$ большое | Сильные циклические/спиральные паттерны |

**Пример:** Для $D=4$:
- Число пар: $4 \times 3/2 = 6$
- `levy_area`: 6 компонент
- `rotation_number`: 1 скаляр

### 5. Дополнительные сигнатурные метрики (детально)

#### 5.1 Энергия по уровням (Signature Energy)

**Формула:**

$$E_k = \lVert S_{\text{level}\,k} \rVert^2 = \sum_{\substack{I:\;\lvert I\rvert = k}} \bigl(S^I\bigr)^2$$

**Интерпретация:**
- $E_1$: Энергия трендов (уровень 1)
- $E_2$: Энергия взаимодействий (уровень 2)
- $E_k / E_{\text{total}}$: Относительный вклад уровня $k$

**Применение:**
- Анализ доминирующих уровней сигнатуры
- Выбор оптимальной глубины `depth`

---

#### 5.2 Энтропия сигнатуры (Signature Entropy)

**Формула:**

$$H(S) = -\sum_{i=1}^{K} p_i \log p_i, \qquad p_i = \frac{\lvert S_i \rvert}{\sum_{j} \lvert S_j \rvert}$$

**Нормализация:**

$$H_{\text{norm}} = \frac{H}{\log K} \in [0,\,1]$$

**Интерпретация:**
- $H \approx 0$: Сигнатура сосредоточена в нескольких компонентах (регулярный процесс)
- $H \approx 1$: Сигнатура равномерно распределена (хаотичный процесс)

---

#### 5.3 Число вращения (Rotation Number)

**Формула:**

$$\mathrm{RN} = \frac{1}{\pi}\sqrt{\sum_{i < j}\bigl(A^{i,j}\bigr)^2}$$

**Интерпретация:**
- $\mathrm{RN} \approx 0$: Нет вращения (прямая траектория)
- $\mathrm{RN} > 0$: Есть циклические паттерны
- $\mathrm{RN} \approx 1$: Одно полное «вращение» за время наблюдения

---

#### 5.4 Расстояние до броуновского движения (Brownian Distance)

**Формула:**

$$d_{\mathrm{BM}} = \lVert S(X) - S(\mathrm{BM}) \rVert$$

где $S(\mathrm{BM})$ — ожидаемая сигнатура стандартного броуновского движения:
- Уровень 1: $\mathbb{E}[S^i] = 0$
- Уровень 2: $\mathbb{E}[S^{i,i}] = T/2$, $\;\mathbb{E}[S^{i,j}] = 0$ для $i \neq j$
- Уровни $3{+}$: $\mathbb{E}[S^I] = 0$

**Интерпретация:**
- $d_{\mathrm{BM}} \approx 0$: Процесс близок к броуновскому движению
- $d_{\mathrm{BM}}$ большое: Процесс имеет структуру (тренды, циклы)

---

#### 5.5 Собственные значения Грама (Gram Eigenvalues)

**Формула:**

$$\mathbf{G} = S \cdot S^\top \in \mathbb{R}^{B \times B}, \qquad \lambda_1 \geq \lambda_2 \geq \cdots \geq \lambda_B \geq 0$$

**Интерпретация:**
- $\lambda_1 \gg \lambda_2$: Данные лежат вблизи одного направления
- $\lambda_1 \approx \lambda_2 \approx \cdots \approx \lambda_k$: Данные равномерно распределены в $k$-мерном подпространстве
- Эффективная размерность: $k$, где $\sum_{i=1}^{k} \lambda_i \,/\, \sum_{j=1}^{B} \lambda_j \approx 0.95$

## Генерация многомерных рядов: как это работает

### Процесс генерации одного многомерного ряда

**Шаг 1.** Генерация временной сетки:

$$t = \text{time\_grid\_sampler}(T,\;\text{mode}=\text{"jitter"},\;t_0=0,\;t_1=100) \;\;\Rightarrow\;\; t \in \mathbb{R}^T$$

**Шаг 2.** Выбор типа латентного процесса:

$$\text{process\_type} \sim \mathrm{Categorical}\bigl(\{\text{arima}: 0.25,\;\text{kernel\_synth}: 0.25,\;\text{ets}: 0.25,\;\text{tsi}: 0.25\}\bigr)$$

**Шаг 3.** Генерация латентных компонент:

$$z = \text{LatentProcess}(t,\,L,\,\theta) \;\;\Rightarrow\;\; z \in \mathbb{R}^{T \times L}$$

**Шаг 4.** Преобразование в наблюдения через вычислительный граф:

$$x_{\text{clean}} = f(z) \;\;\Rightarrow\;\; x_{\text{clean}} \in \mathbb{R}^{T \times D}$$

Пример графа:

$$x_{:,\,0} = \sin(z_{:,\,0}) \cdot z_{:,\,1} + z_{:,\,2}$$
$$x_{:,\,1} = \tanh(z_{:,\,0} + z_{:,\,1}) \cdot \exp(-z_{:,\,2})$$

**Шаг 5.** Временные сдвиги:

$$\forall\, k \in \{1,\ldots,L\}:\quad \delta_k \sim \mathrm{Uniform}(-20,\,20), \quad z_{\text{shifted}}^{(k)} = \mathrm{roll}(z^{(k)},\;\delta_k)$$

**Шаг 6.** Добавление шума:

$$\varepsilon_{\text{add}} \sim t_\nu(0,\,\sigma_{\text{add}}), \qquad \eta_{\text{mult}} \sim \mathrm{Weibull}(k,\,\lambda)$$
$$x_{\text{noisy}} = x_{\text{clean}} \cdot \eta_{\text{mult}} + \varepsilon_{\text{add}}$$

**Шаг 7.** Финальная нормализация:

$$x = \frac{x_{\text{noisy}} - \bar{x}_{\text{noisy}}}{\mathrm{std}(x_{\text{noisy}})} \;\;\Rightarrow\;\; x \in \mathbb{R}^{T \times D}$$
